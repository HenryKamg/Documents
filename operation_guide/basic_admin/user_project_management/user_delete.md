# 删除用户

### 通过Web horizon删除用户

* 登录Web horizon界面----点击identity-----点击用户----删除用户

![User_Delete](/operation_guide/basic_admin/Picture/user_delete1.jpg)


### 通过命令删除用户

* 删除用户，执行如下命令

> ``` keystone user-delete name```
