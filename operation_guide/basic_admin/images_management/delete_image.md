# 删除模板

### 通过Web horizon界面删除模版

* 登录Web horizon界面，选择镜像，选中需要删除，点击删除镜像

![Image-Delete](operation_guide/basic_admin/Picture/delete_image1.jpg)


### 通过命令模版

* 删除模版，执行如下命令

> ```glance image-delete NAME```
