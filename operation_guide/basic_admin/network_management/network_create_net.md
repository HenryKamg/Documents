# 创建内部网络

### 通过Web horizon创建内部网络

* 登录Web horizon点击管理员----系统----网络----创建网络

![Networkext_Create](../Picture/network_createnet1.jpg)

> 填写名称、选择项目、供应商类型、点击创建网络

* 点击新建的网络，进入子网创建界面

![Networkext_Create](../Picture/network_createnet2.jpg)

* 点击下一步，进入子网详情页面，填写信息

![Networkext_Create](../Picture/network_createnet3.jpg)

* 点击已创建，创建完成
