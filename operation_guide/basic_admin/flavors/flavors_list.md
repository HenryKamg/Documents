# 查看已有规格

### 通过We horizon界面查规格

* 登录Web horizon界面，点击```云主机类型```显示规格信息

![Flavors_List](/operation_guide/basic_admin/Picture/flavors_list.jpg)

云主机类型```Public 为Yes```表示所有用户都能使用

已存在的云主机类型，可以通过```编辑云主机类型```调整云主机类型信息


### 通过命令查看云主机规

* 查看云主机规格，执行如下命令

> ```nova flavor-list```
