# Fuel节点管理命令

该命令用于对Fuel节点进行备份与恢复。

## 命令格式

```
[fuel]$ eayunstack fuel --help
usage: eayunstack fuel [-h] COMMAND ...

EayunStack Fuel Management

optional arguments:
  -h, --help  show this help message and exit

Commands:
  COMMAND     DESCRIPTION
    restore   Fuel Restore
    backup    Fuel Backup
```
