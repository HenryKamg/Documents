# 安装插件

## 安装Cinder Eqlx插件

* 登录Fuel节点，执行以下命令，安装Cinder Eqlx插件。

 ```
# fuel plugins --install /opt/eayunstack/cinder_eqlx-1.0.0.fp
```

## 安装Neutron LBaaS插件

* 登录Fuel节点，执行以下命令，安装Neutron LBaaS插件。

